Coverage

What CITADEL checks

Ten analysis engines run over your code in a single pass, and every finding is cross-walked to the controls of 20+ compliance standards. Here's the full coverage.

Analyze something now

Analysis engines

Each engine feeds the same framework-aware report.

SAST

280+ heuristic rules (plus Semgrep/Bandit on the backend) for injection, XSS, crypto, deserialization, SSRF, traversal & more.

Secrets

Entropy + pattern detection for hardcoded keys, tokens & credentials (Gitleaks/Trivy on the backend).

PII & sensitive data

Detects exposed personal/regulated data — SSNs, payment cards, IBANs, health (PHI) identifiers, JWTs — mapped to GDPR, HIPAA, PCI DSS & SOC 2.

SBOM & live CVEs

CycloneDX SBOM from your manifests, cross-checked against the OSV.dev advisory database for real CVEs.

Binary analysis

PE/ELF/Mach-O detection, entropy/packing, string extraction & suspicious-capability indicators (ClamAV on the backend).

Compliance mapping

Every finding cross-walked to the exact controls it implicates across 20+ frameworks.

Quality & licenses

Maintainability index, comment density, oversized files, and license detection (flags copyleft).

Deployment detection

Infers how the project ships: Docker, Kubernetes, Helm, Terraform, Bicep, CI/CD, PaaS.

AI remediation

Per-finding "Explain & fix" powered by Claude when the backend is configured with an API key.

Exports & CI

JSON, SARIF (code scanning), CycloneDX, POA&M, SSP appendix, Markdown, PDF — plus a CLI & GitHub Action.
