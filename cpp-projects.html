// CPP-001 — ENTROPY-SCANNER

Binary Entropy Scanner

Detects packed, encrypted, and obfuscated binaries by computing Shannon entropy per file. High entropy (≥7.2 bits) strongly indicates AV-evasion techniques or ransomware encryption.

  • Shannon entropy (0–8 scale) per file
  • Recursive directory scanning
  • Verdicts: PACKED/ENCRYPTED · COMPRESSED · PLAINTEXT
  • Exit code 2 on any high-entropy findings
g++ -std=c++17 -O2 entropy_scanner.cpp -o entropy-scanner

Deployment Guide

C++ must be compiled before it runs. Choose the deployment path that matches your target environment.

🐧 Linux / macOS — Native Binary

Fastest path. Produces a native binary that runs directly on your system. Recommended for all security tooling and any project marked Native Only.

1
Install a C++17 compiler
Ubuntu/Debian: sudo apt install g++ build-essential
macOS: xcode-select --install (installs Clang)
Fedora/RHEL: sudo dnf install gcc-c++
2
Install OpenSSL (for AES Vault only)
Ubuntu: sudo apt install libssl-dev
macOS: brew install openssl then add -I$(brew --prefix openssl)/include -L$(brew --prefix openssl)/lib
3
Compile and run
# Clone or download the project cd cpp/entropy-scanner # Compile (replace with project-specific command) g++ -std=c++17 -O2 entropy_scanner.cpp -o entropy-scanner # Run ./entropy-scanner /path/to/scan
4
Strip and deploy
# Reduce binary size strip entropy-scanner # Install system-wide (optional) sudo cp entropy-scanner /usr/local/bin/

📼 Windows — MSVC or MinGW

Two paths: Visual Studio (MSVC) for best Windows integration, or MinGW-w64 for GCC-compatible builds. Projects marked Linux Only cannot run on Windows.

1
Install Visual Studio 2022 (MSVC)
Download Visual Studio Community (free). Select "Desktop development with C++" workload. This installs cl.exe, MSVC stdlib, and the Windows SDK.
2
Or install MinGW-w64 (GCC for Windows)
Install via winget install GnuWin32.GnuMake or download MSYS2 (msys2.org) and run: pacman -S mingw-w64-x86_64-gcc
3
Compile (Developer Command Prompt)
:: MSVC cl /std:c++17 /O2 /EHsc entropy_scanner.cpp /Fe:entropy-scanner.exe :: MinGW (in MSYS2 shell) g++ -std=c++17 -O2 entropy_scanner.cpp -o entropy-scanner.exe
4
OpenSSL on Windows (AES Vault)
Install via: winget install ShiningLight.OpenSSL
Then add to link flags: /I "C:\Program Files\OpenSSL-Win64\include" libssl.lib libcrypto.lib

🌐 WebAssembly — Run in the Browser

Compile C++ to .wasm using Emscripten, then host on GitHub Pages. Only projects marked WASM ✓ are compatible (no OS-level system calls).

1
Install Emscripten SDK
git clone https://github.com/emscripten-core/emsdk.git cd emsdk ./emsdk install latest ./emsdk activate latest source ./emsdk_env.sh # Linux/macOS emsdk_env.bat # Windows
2
Compile to WASM
em++ -std=c++17 -O2 entropy_scanner.cpp \ -o entropy_scanner.js \ -s EXPORTED_FUNCTIONS='["_main"]' \ -s ALLOW_MEMORY_GROWTH=1 \ -s MODULARIZE=1 \ --bind
3
Host on GitHub Pages
Copy the generated .js and .wasm files to your repo. Load the JS module in an HTML page. GitHub Pages serves .wasm with correct MIME type (application/wasm) automatically.
4
Why WASM limitations exist
Projects using raw sockets, /proc, ptrace, or hardware I/O cannot compile to WASM — the browser sandbox blocks OS-level calls. Projects marked Native Only fall into this category.

🏗 Docker — Reproducible Build Container

Best for CI/CD, sharing pre-built binaries, or running Linux-only tools on Windows/macOS without dual-booting.

1
Dockerfile (multi-stage build)
FROM gcc:13 AS builder WORKDIR /src COPY . . RUN g++ -std=c++17 -O2 -static \ entropy_scanner.cpp -o entropy-scanner FROM debian:bookworm-slim COPY --from=builder /src/entropy-scanner /usr/local/bin/ ENTRYPOINT ["entropy-scanner"]
2
Build and run
docker build -t entropy-scanner . docker run --rm -v $(pwd)/files:/scan \ entropy-scanner /scan
3
Memory scanner (requires privileged)
docker run --rm --privileged \ --pid=host \ memory-scanner 1234
4
GitHub Actions CI build
- name: Build C++ tools run: | sudo apt-get install -y libssl-dev for dir in cpp/*/; do cd "$dir" && g++ -std=c++17 -O2 *.cpp \ -lssl -lcrypto -o "$(basename $dir)" \ 2>/dev/null || g++ -std=c++17 -O2 *.cpp \ -o "$(basename $dir)"; cd ../.. done
