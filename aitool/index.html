Regulatory Quick Reference
CMMC 2.0

Cybersecurity Maturity Model Certification. Level 2 requires implementation of all 110 NIST SP 800-171 practices. AI tools touching CUI must be assessed under the CMMC framework.

  • 14 practice domains
  • 110 practices at Level 2
  • 130 practices at Level 3
  • Triennial C3PAO assessment (L2)
NIST 800-171

Protecting Controlled Unclassified Information (CUI) in Non-Federal Systems. Mandatory for DoD contractors under DFARS clause 252.204-7012.

  • 14 requirement families
  • 110 security requirements
  • Self-assessment via SPRS
  • Rev 3 (2024) alignment
ISO 27001:2022

International standard for Information Security Management Systems. Annex A contains 93 controls across 4 themes. Key for AI: cloud security, supplier relationships.

  • 4 themes, 93 controls
  • A.5.23: Cloud services
  • A.5.19–5.22: Supplier mgmt
  • A.8.25: Secure dev lifecycle
DFARS 7012

Safeguarding Covered Defense Information (CDI) and reporting cyber incidents. Cloud services used to process CDI must meet FedRAMP Moderate or equivalent security requirements.

  • 72-hour cyber incident reporting
  • FedRAMP Moderate for cloud
  • Malicious code isolation
  • Media preservation & forensics
